### Q. Call By Value와 Call By Reference 차이
함수를 호출할 때 매개변수로 전달되는 값의 종류에 따라 call by value, call by reference로 나뉩니다. value는 원본 값을 그대로 복사하므로, 원본에 영향을 미칠 수 없습니다. 반면에 reference는 원본의 주소값을 복사하여 전달하므로, 원본에 영향을 줄 수 있습니다.
